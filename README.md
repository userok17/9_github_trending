# Github Trends

Скрипт "Github Trends" находит все репозитории, созданные за последнюю неделю, выберает 20 с наибольшим количеством звёзд и для каждого посчитатывает количество открытых issues.

# Как запустить

Скрипт требует для своей работы установленного интерпретатора Python версии 3.5

Запуск на Linux:

```#!bash
git clone https://github.com/userok17/9_github_trending

pip3 install -r requirements.txt

python3 github_trending.py
```
